# 신의 한수 the 오목
자신의 계정을 이용해 어플 내 사용자와 PvP 대전이 가능한 오목 어플리케이션입니다
## ⭐ 핵심 기능 
### 주요 기능
* 실시간 2인 PvP 대전 
* 오목 게임 방 개설 
* 개인 계정 로그인 (회원가입, 로그인)
### 부가 기능
* 게임 내 배경 음악 켜기/끄기
